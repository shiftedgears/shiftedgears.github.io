Network Printer Sharing Windows 11

Firstly, I am not taking credit for this process. I just did some digging and found this in a help thread, and thought it would be good to make a copy of it so others can more easily find it. Shout out to the Anon on the Windows support thread that laid all of this out - he's the real hero.

This is a guide on how to share a printer that is connected via USB over the local network to other computers on the network. This guide requires sharing login credentials to another users computer, so the proper way to do this would be to create a seperate user that explicitly is for printing that other users can "login" as. I am not doing that in this guide because I am lazy.

  1. First, ensure both PCs are connected to the same netowork.
  2. Set the network profile for both PCs to "Private Network."
  3. Allow "File and Printer Sharing" on both PCs' Firewall settings
  4. On the Main PC, the one connected to the USB printer, open "Advanced Sharing Settings." Under "All Networks," enable "Password protected sharing."
  5. To share the printer, on the Main PC go to Printers & Scanners > the printer > Printer Properties > Sharing > and tick the Share this printer option whilst giving it a discoverable name.
  6. On the Sub PC, the PC that is connecting via the network, open Credential Manager and click on Windows Credentials.
  7. Under Windows Credentials, click on "Add a Windows credential."
  8. You will see three input fields: Internet or network address, User name, and Password.
    1. Internet or network address: enter the ip address or PC name of the Main PC. Ensure you use a double back slack before the address (\\192.168.X.X or \\DESKTOP-XXXXXXX)
    2. User name: Enter the account name of the Main BC. This account must have Administrator Access.
    3. Password:Enter the password of the account name.
  9. On the Sub PC open the Network folder. Turn on network discovery and file sharing.
  10. Locate the Main PC's name. Double click on it, and you will see the shared printer.
  11. Right-click on that printer and click "open." The printer will be installed.
