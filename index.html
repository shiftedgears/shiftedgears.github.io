Projects

Image Board (Tekku)

In the fourth year of my CSCI degree some colleagues (Ethan Posner and Spencer Muise) and myself made an online image board using html/css/PHP/minimal JS, very much in the style of the Chan/Booru's of the early internet. Depending on if we are currently hosting it or not you can go to it at tekku.site. This site included full user profile support, password hashing with salt+pepper, file storing, image hosting, etc. You can view the github page for it here.
