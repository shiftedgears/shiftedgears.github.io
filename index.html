himark Chat

In my final year I worked on a project with Curtis Bachen and Vladislav Mazur in which we created a command line based chat application in Python. This was mainly done through Aysncio and Websockets. It is comprised of two seperate applications: the server and the client programs. A user runs the the server software on the computer they wish to host their server, listening on port 8000 by default. Another user will run the client software and attempt to connect to a server being hosted on the local machine by default. The source code for each can be tweaked to edit the port/ip. The server has a text file with a list of room names, and creates a room for each given name. Users can connect to different rooms and swap between rooms at any time. They can also see whoever else is in the room at any given time. This was made using asyncio, websockets, and textual for the great CLI. You can view the github page for it here.
